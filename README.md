## Project Overview

> **CyberMan** is an AI powered cybersecurity assistant built using Python in a Jupyter Notebook environment.  
> The chatbot provides high-detail technical answers, programming insights, and cybersecurity explanations through an interactive system.

The project demonstrates how AI can function as a **technical knowledge assistant for cybersecurity and computing concepts**.

## Installation

### Clone Repository

```
git clone https://github.com/yourusername/CyberMan.git
```

### Go to Project Folder

```
cd CyberMan
```

### Run Jupyter Notebook

```
jupyter notebook
```

Open **CyberMan.ipynb** and run all cells.
